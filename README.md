A React-based e-commerce platform that demonstrates advanced state management and routing concepts.  API data integration (FakeStore API)  Dynamic routing with React Router  Protected admin routes  Product CRUD functionality  Cart system with quantity controls  Data persistence via localStorage  Theme switching (Light/Dark mode)  Responsive UI

## Features
- Product listing
- Category filtering
- Product details
- Cart system
- Admin product management
- Authentication
- Dark mode

## Tech Stack
- React
- React Router
- Tailwind CSS
- FakeStore API
- LocalStorage

## Installation
npm install
npm run dev
